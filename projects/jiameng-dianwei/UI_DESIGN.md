# 🎨 奶茶选址通 — UI设计规范
> 版本：v1.0 | 日期：2026-03-31 | 状态：已确认

## 1. 设计风格
> 一句话：暖橙色调的专业数据平台，简洁不花哨，让加盟商觉得靠谱

## 2. 配色（方案A：奶茶暖橙）
| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#E8751A` | 按钮、链接、重点 |
| 辅色 | `#F5A623` | 强调、标签、图标 |
| 背景 | `#FFF8F0` | 页面背景 |
| 卡片背景 | `#FFFFFF` | 卡片、弹窗 |
| 文字主色 | `#1A1A1A` | 正文 |
| 文字次色 | `#8C8C8C` | 说明文字 |
| 成功色 | `#22C55E` | 正面状态 |
| 警告色 | `#F59E0B` | 注意状态 |
| 错误色 | `#EF4444` | 错误状态 |

## 3. 字体
- 标题/正文：PingFang SC / Microsoft YaHei（系统字体）
- 数字/价格：DIN Alternate / Roboto

| 用途 | 字号 | 字重 |
|------|------|------|
| 大标题 H1 | 36px | Bold |
| 小标题 H2 | 24px | Semibold |
| 三级标题 H3 | 20px | Semibold |
| 正文 | 16px | Regular |
| 小字/说明 | 14px | Regular |
| 极小字 | 12px | Regular |
| 价格数字 | 32px | Bold |

## 4. 间距规范
- 页面边距：16px（手机）/ 32px（平板）/ 64px（电脑）
- 卡片间距：16px
- 卡片内边距：16px（手机）/ 24px（电脑）
- 按钮圆角：8px
- 卡片圆角：12px
- 输入框圆角：8px

## 5. 组件样式

### 按钮
- 主按钮：`#E8751A` 背景 + 白色文字 + 8px圆角，hover → `#D06515`
- 次按钮：白色背景 + `#E8751A` 边框和文字，hover → `#FFF8F0` 背景
- 文字按钮：无背景 + `#E8751A` 文字，hover → 下划线
- 禁用：`#CCCCCC` 背景 + 白色文字

### 商圈卡片
- 白色背景 + 12px圆角 + 阴影 `0 2px 8px rgba(0,0,0,0.08)`
- 包含：商圈名称、人流量、奶茶/咖啡数量预览、两个购买按钮、更新时间

### 价格标签
- 基础报告：橙色底 `#FFF3E8` + 橙色字 `#E8751A`
- AI研判：渐变底 `#E8751A → #F5A623` + 白色字

### 数据卡片
- 大号数字 32px Bold `#E8751A` + 说明文字 14px `#8C8C8C`

### 品牌标签
- 圆角全圆 + 品牌色底 + 对应色文字
- 蜜雪冰城(浅红)、瑞幸(浅蓝)、喜茶(浅绿)、其他(浅灰)

### 输入框
- 边框 `#E5E5E5` + 8px圆角，focus → `#E8751A` 边框

## 6. 页面清单
1. 首页/落地页 — 产品介绍 + CTA + 热门商圈
2. 注册/登录页 — 手机号+验证码 / 微信登录
3. 城市选择页 — 城市列表（MVP只有金华）
4. 商圈列表页 — 搜索 + 筛选 + 商圈卡片列表
5. 商圈详情页(未付费) — 免费信息 + 模糊数据 + 付费按钮
6. 商圈详情页(基础报告) — 竞品数据 + 品牌分布 + 地图 + 升级AI入口
7. AI研判报告页 — 评分仪表盘 + 品牌推荐/预警 + 风险 + 建议
8. 支付页 — 产品选择 + 支付方式
9. 我的报告页 — 已购报告列表
10. 个人中心 — 用户信息

## 7. 响应式断点
| 设备 | 宽度 | 布局变化 |
|------|------|---------|
| 手机 | < 768px | 单列，卡片堆叠，底部导航 |
| 平板 | 768-1024px | 双列商圈卡片 |
| 电脑 | > 1024px | 三列商圈卡片，侧边筛选栏 |
