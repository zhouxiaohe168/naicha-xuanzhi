# LEARN — 一人公司学习加速器

帮助用户用最短时间掌握新技能，只学"够用"的部分。

## 核心理念

> 一人公司创业者不需要成为专家，只需要"够用+AI辅助"就能产出专业级成果。
> 学80%最常用的20%知识就够了。

## 当用户说 /learn [技能] 时

### Step 1: 快速评估

先问用户两个问题：
1. **你学这个是为了什么？**（做哪个项目/解决什么问题）
2. **你现在是什么水平？**（完全零基础 / 知道一点 / 用过但不熟）

### Step 2: 生成"够用"学习路径

根据用户的目的，只列出必须学的内容：

```
学习路径：[技能名称]
目标：[够用标准 — 能做到什么就够了]
预计时间：[X天/X周]

第1天：[最基础的概念 + 动手练习]
第2天：[核心技能1 + 小项目实践]
第3天：[核心技能2 + 小项目实践]
...
毕业项目：[用学到的东西做一个真实的小产品]
```

### Step 3: 推荐资源（最多3个）

- **首选**：免费的、中文的、实战导向的
- **次选**：优质的付费课程（如果免费资源不够好）
- **工具**：可以辅助学习的AI工具

**重要**：不要推荐超过3个资源，选择太多反而导致不行动。

### Step 4: 给出"今天就开始"的第一步

一个15分钟内能完成的具体任务，降低启动门槛。

## 学习方法论

### 费曼学习法（最适合新手）
```
学一个概念 → 用自己的话说出来 → 说不清楚的地方就是没懂 → 回去重学
```

### "边做边学"优于"先学再做"
```
不要：看完100集教程 → 开始做项目
要：看10分钟 → 动手试 → 卡住了再学 → 继续做
```

### AI辅助学习
```
遇到不懂的 → 问Claude/ChatGPT → 用自己的话复述 → 动手验证
这比看教程快5倍。
```
