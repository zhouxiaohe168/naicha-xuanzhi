# CONTENT — 一人公司内容引擎

帮助用户快速创作高质量内容，建立个人品牌。

## 内容是一人公司最重要的杠杆
> 写一次 → 永远在为你工作 → 吸引客户 → 建立信任 → 产生收入

## 当用户说 /content 时

### Step 1: 确认内容类型

| 类型 | 格式 | 适合平台 |
|------|------|----------|
| **推文/帖子** | 280字以内，配图建议 | Twitter/X、即刻 |
| **长推文/Thread** | 5-10条串联推文 | Twitter/X |
| **技术文章** | 1000-2000字，有代码示例 | 公众号、掘金、Medium |
| **教程** | 分步骤，有截图说明 | 博客、YouTube |
| **视频脚本** | 开头hook+正文+CTA | YouTube、B站 |
| **Newsletter** | 本周精选+个人洞察 | Substack、竹白 |

### Step 2: 生成内容

**推文模板**（直接生成可发布的内容）：
```
Hook（吸引注意力的第一句）
↓
价值点（用户能学到什么）
↓
个人观点（你的独特见解）
↓
CTA（引导互动：你觉得呢？/ 关注获取更多）
```

**文章模板**：
```
标题（数字+痛点+结果）
↓
开头（一个故事或问题引入）
↓
正文（3-5个要点，每个有例子）
↓
总结（核心要点回顾）
↓
CTA（关注/订阅/评论引导）
```

### Step 3: 优化建议
- 标题是否有吸引力？给出3个备选标题
- 开头前30字是否能留住读者？
- 是否有可分享的金句？

## 内容策略建议

### 80/20内容比例
- **80% 价值内容**：教程、洞察、工具推荐、经验分享
- **20% 个人/产品内容**：创业故事、产品更新、里程碑

### 内容复用（一鱼多吃）
```
一篇长文章
  ├→ 拆成5条推文
  ├→ 提炼成1张信息图
  ├→ 改编成1个短视频脚本
  └→ 作为Newsletter的一期
```

## 写作风格指南

- **口语化** — 像跟朋友聊天，不要学术腔
- **有观点** — 不做中立的搬运工，要有自己的判断
- **有故事** — 用亲身经历/案例支撑观点
- **有数据** — 用数字增加说服力
- **简短** — 一段不超过3行，多用列表和小标题
