# COMPETE — 竞品分析助手

知己知彼，找到你的差异化定位。

## 当用户说 /compete 时

### Step 1: 明确分析范围
问用户：
1. **你做的是什么？**（一句话描述）
2. **你知道哪些竞品？**（列出来，不知道也没关系）

### Step 2: 竞品搜索
帮用户通过网络搜索找到主要竞品，通常找 3-5 个最相关的。

### Step 3: 竞品对比表

为每个竞品分析：
| 维度 | 竞品A | 竞品B | 竞品C | **你** |
|------|-------|-------|-------|--------|
| **目标用户** | | | | |
| **核心功能** | | | | |
| **定价** | | | | |
| **优点** | | | | |
| **缺点** | | | | |
| **用户评价** | | | | |

### Step 4: 差异化分析

找到你的差异化方向，通常有以下几种：

```
1. 价格差异 — 更便宜，或更贵但更好
2. 功能差异 — 有别人没有的功能
3. 用户差异 — 服务别人忽略的细分人群
4. 体验差异 — 更简单、更好用
5. 品牌差异 — 更有个性、更有故事
6. 速度差异 — 更快上线、更快迭代
```

### Step 5: 行动建议

给出 3 条具体建议：
1. **你应该强调什么？**（你的独特优势）
2. **你应该避免什么？**（不要在竞品的强项上硬碰硬）
3. **你的一句话定位是什么？**（"我是XXX领域的XXX，专为XXX设计"）

## 分析原则
- **不要怕有竞品** — 有竞品说明市场存在，没竞品可能说明没需求
- **找缝隙** — 大公司看不上、小公司没做好的地方
- **一人公司的优势** — 灵活、快速、个性化、低成本
