# PITCH — 一句话说清楚你是谁

一人公司最重要的能力之一：让别人30秒内理解你做什么、为什么重要。

## 当用户说 /pitch 时

### Step 1: 确认场景

| 场景 | 长度 | 目的 |
|------|------|------|
| **Twitter/社交简介** | 1-2句话 | 让人想关注你 |
| **电梯演讲** | 30秒/60秒 | 让人记住你 |
| **客户介绍** | 1分钟 | 让人想买你的产品 |
| **合作邀约** | 2-3段 | 让人想和你合作 |
| **投资人pitch** | 3分钟 | 让人想投你 |

### Step 2: 收集信息

问用户：
1. **你是谁？**（名字/身份）
2. **你做什么？**（核心产品/服务）
3. **为谁做？**（目标用户）
4. **有什么成果？**（数据/案例，没有也行）
5. **你的愿景是什么？**（为什么做这件事）

### Step 3: 生成Pitch

**Twitter简介公式**：
```
[身份] | [在做的事] | [成果/愿景] | [个性标签]
```
示例：一人公司创业者 | 用AI打造量化交易系统 | 让普通人也能做专业交易 | 代码+咖啡驱动

**电梯演讲公式（30秒）**：
```
你知道[目标用户的痛点]吗？
我做了[你的产品/方案]，
它能[核心价值/解决方案]。
目前已经[成果/进展]。
我正在[下一步计划/寻求的帮助]。
```

**客户介绍公式**：
```
[痛点共鸣] — 你是不是也遇到了...
[方案介绍] — 我做了一个...
[差异化] — 和其他方案不同的是...
[社会证明] — 已经有XX人在用...
[CTA] — 你可以...开始试用
```

### Step 4: 给出3个版本

- **版本A**：专业正式版
- **版本B**：轻松有趣版
- **版本C**：数据驱动版

让用户选一个最像自己的风格。

## 原则
- **说人话** — 不要用行业黑话
- **有画面感** — 让人脑子里能想象出场景
- **有记忆点** — 至少有一句话让人记住
- **有行动** — 说完之后对方知道下一步做什么
